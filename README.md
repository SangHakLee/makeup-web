## 📸 이런 경험, 한 번쯤 있지 않나요?

공들여 만든 사이트를 카카오톡이나 슬랙에 공유했는데...

- 🖼️ **썸네일**이 안 뜬다
- ✂️ **제목**이 이상하게 잘려 나온다
- 📄 **설명**이 아예 비어있거나 깨져 있다

웹사이트에 접속하기도 전에, 첫인상은 이미 망가져 버렸을지도 모릅니다. (소개팅 전에 막 일어난 얼굴 셀카를 보낸다고 생각해 보세요! 😱)

**Makeup Web**은 바로 이런 문제를 해결합니다. '딸깍' 한 번으로 만든 서비스라도 기본적인 미(美)를 갖출 수 있도록, URL 하나로 웹사이트의 미모를 평가하고 부족한 점을 고쳐 나갑니다.

---

## ✨ 주요 기능

### 1. 외적인 아름다움 가꾸기 (Link Preview Diagnosis)
URL을 공유했을 때 보여지는 첫인상을 진단합니다.
- **OG(Open Graph)** 태그 분석: 썸네일, 제목, 설명이 제대로 설정되어 있는지 확인합니다.
- **미리보기 시뮬레이션**: 다양한 플랫폼(카카오톡, 페이스북 등)에서 어떻게 보일지 미리 확인합니다.

### 2. 내면의 아름다움 채우기 (SEO Analysis)
겉모습만 예쁘다고 끝이 아닙니다. 검색 엔진이 사랑하는 웹사이트가 되어야 합니다.
- **검색 엔진 최적화(SEO) 평가**: 웹사이트가 검색 로봇에게 얼마나 효율적으로 노출되는지 분석합니다.
- **구조적 개선 제안**: 더 넓은 세상으로 나아가기 위한 내실을 다져줍니다.

### 3. AI 기반의 퍼스널 컬러 컨설팅 (AI Proposals)
진단에서 멈추지 않고, AI가 직접 개선 방법을 제안합니다.
- **브랜딩 자산 생성**: 웹사이트의 성격을 파악해 어울리는 **파비콘(Favicon)**, **아이콘**, **OG Image**를 직접 만들어 드립니다.
- **콘텐츠 최적화**: 방문자의 클릭을 부르는 매력적인 **웹사이트 설명(Description)** 문구를 제안합니다.
